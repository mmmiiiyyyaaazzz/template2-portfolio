はい。ZoomやGoogle Meetで対応可能です。機密保持のため、録画は原則行いません。

ご相談の流れ

初回のご相談から着手までのステップです。オンライン面談も可能です。

  1. お問い合わせ

    フォームまたはお電話よりご連絡ください。日程調整の上、初回面談をご案内します。

  2. ヒアリング・課題整理

    現状やご希望を伺い、リスクや対応方針を整理します。（オンライン可）

  3. ご提案・お見積もり

    進め方・期間・費用をご提案。必要に応じてスコープ調整を行います。

  4. ご契約・着手

    ご同意後に着手。継続的な伴走支援にも対応します。
